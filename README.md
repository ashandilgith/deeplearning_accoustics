# Piranaware V2: Acoustic Anomaly Detection

This project uses a Convolutional Autoencoder to detect engine anomalies.

**How to use:**
1. Go to the **Training** tab.
2. Record "Normal" audio for Idle, Slow, and Fast speeds.
3. Switch to **Diagnostics** to test new sounds.
